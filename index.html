HOSPITAL MANAGEMENT

A desktop application made using Java. The specific library used was Java Swing for the GUI. This application allows administrators to compose schedules for doctors as well as schedule/reschedule patients in for their appointments. Within the same app, doctors can request time off and input their availabilities so that the administrator can process this information and make their schedule for them. Doctors can then view their schedules. Patients are able to view their appointments after the administrator books them in.
